🚀 Real sign-in + payments setup guide
Step 1 — Netlify (free hosting)
Drag your index.html to netlify.com → you get a real https:// URL in minutes. Custom domain ~$15/yr optional.
Step 2 — Firebase Auth (free Google/Yahoo login)
firebase.google.com → create project → Authentication → enable Google + Yahoo providers → paste the 10-line JS snippet I write for you into your HTML. Apple login needs an Apple Developer account ($99/yr).
Step 3 — Stripe + Firebase Extension (real payments)
stripe.com → create account → add your 4 products ($0/$9/$12/$18/mo) → Firebase console → Extensions → "Run Payments with Stripe" → install. 2.9% + 30¢ per transaction, no monthly fee.
Step 4 — Upload your Pascal code here
Attach .pas / .dpr / .dfm files in this chat. I'll translate your logic into the wired-up web version and hand you deploy-ready code to drag back to Netlify.
Total minimum launch cost: $0 (skip Apple login + custom domain). Stripe fees only apply when you earn revenue.
